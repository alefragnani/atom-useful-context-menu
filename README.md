# Useful Context Menu Package for Atom

A set of useful context menu actions for Atom

## Available Actions

By now, I have added a few actions

#### Copy to Clipboard

- Copy File Path
- Copy Filename
- Copy Folder Path

#### Open In

- Open In File Manager
- Open In Terminal
